# 📰 Card-News Generator

내부 커뮤니케이션용 카드뉴스 블랭크 템플릿 생성기  
SK 사내 에디토리얼 스타일 기반 · 1080 × 1350px · PNG 완성본 출력

🔗 **배포 URL:** https://yena001.vercel.app

---

## 버전 히스토리

| 버전 | 파일명 | 주요 기능 | 날짜 |
|------|--------|-----------|------|
| v1 | `v1-blank-template.html` | 빈 템플릿 생성기 (텍스트 없음) · 가이드 오버레이 · PNG 내보내기 | 2025-05 |
| v2 | `v2-text-input.html` | 텍스트 직접 입력 · 실시간 캔버스 반영 · 동적 항목 추가/삭제 | 2025-05 |
| v3 | `v3-autofit.html` | Auto-fit 텍스트 · 글자 크기 슬라이더 · 정렬 선택 · 컬러피커 · 다크배경 대응 | 2025-05 |
| 최신 | `index.html` | 현재 Vercel 배포 버전 (항상 최신) | - |

## 템플릿 종류

- **① 커버** — 챕터 라벨 / 헤드라인 / 서브타이틀 / 본문 / 푸터
- **② 본문** — 긴 텍스트 중심 레이아웃
- **③ 체크리스트** — 항목 2~8개 동적 조절
- **④ 인용 / 핵심 메시지** — 강조 프레임 레이아웃
- **⑤ 데이터 인사이트** — 이미지/차트 영역 포함
- **⑥ 마무리 요약** — 요약 포인트 2~6개 동적 조절

---

## 주요 기능

- ✨ **Auto-fit** — 텍스트가 영역을 넘치면 글자 크기 자동 축소
- 🔤 **글꼴 선택** — Pretendard / Paperlogy / Noto Sans KR 등 6종
- 🎨 **컬러피커** — 강조색 · 배경색 직접 선택 (HEX 입력 가능)
- 📐 **정렬 선택** — 영역별 왼쪽 / 가운데 정렬
- 📏 **슬라이더** — 글자 크기 · 줄간격 · 여백 · 모서리 반경 조절
- 💾 **PNG 내보내기** — 가이드 자동 제거 후 완성본 저장
